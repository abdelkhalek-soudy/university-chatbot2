## 📌 Overview

This project is a smart chatbot built using React + Flask + OpenAI APIs. It is designed to assist students of Badya University by answering their academic questions and analyzing various file types (PDFs, Word documents, images, and audio).

---

## 🧠 Features

- 🔐 **Login Authentication** (with JWT)
- 💬 **Text-based Chat** with GPT-4o
- 🖼️ **Image OCR** using Tesseract (Arabic + English)
- 🗣️ **Voice-to-text** using OpenAI Whisper
- 📄 **File Analyzer** (PDF, DOCX, RTF, JPG, PNG, MP3, WAV, WebM)
- 📦 Fully containerized with React frontend + Flask backend
- 🏷️ **Intelligent filtering** to verify that uploaded content is related to the university
- 🌍 **Multilingual UI** toggle (Arabic / English)
- 👨‍🎓 Personalized chatbot interface with student name and avatar
